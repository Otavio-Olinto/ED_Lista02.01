# ED_Lista02.01
## Produto: Soma Recursiva

Este desafio consiste em criar uma aplicação, em Java, que apresente o produto entre 2 números naturais.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A produto deve ser realizado por somas através de uma função recursiva;
3. Declarar, via comentário, a condição de parada da função.
```
